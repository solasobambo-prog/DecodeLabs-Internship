## 📊 Core Project Summaries

### 🛠️ Project 1: Data Cleaning Verification Gate & Changelog Audit
*   **The Deliverables:** Production-ready cleaned dataset (`.csv`) and a comprehensive Data Quality Changelog Audit Report (`.pdf`).
*   **The Solution:** Executed a structured validation process establishing a strict "Verification Gate" to audit raw data transitions. Documented every anomaly correction step, text parsing adjustment, and duplicate elimination process in an executive changelog report to ensure absolute data transparency and audit-ready integrity.


### 📉 Project 2: Multi-Year Revenue Integrity & Trend Analysis (EDA)
*   **The Problem:** Over 1,200 static transaction records across a three-year period (2023–2025) contained severe data-entry outliers and untracked revenue leakages.
*   **The Solution:** Executed a deep Exploratory Data Analysis comparing statistical distributions (Mean vs. Median anomalies) and isolated 62 severe outlier errors. Uncovered a critical 41% order-to-revenue leak caused by cancellations/returns and calculated platform-specific conversion metrics proving Instagram's superior profitability over Facebook.
